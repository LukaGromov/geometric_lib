# Geometric Lib

## Общее описание решения

Эта библиотека предоставляет функции для вычисления площади и периметра различных геометрических фигур. На данный момент поддерживаются круг и квадрат.

Библиотека реализована на Python и состоит из двух модулей:
- `circle.py` - функции для работы с кругом
- `square.py` - функции для работы с квадратом

## Описание функций

### Модуль circle.py

#### area(r)
Вычисляет площадь круга.

**Параметры:**
- r (float) - радиус круга

**Возвращает:**
- float - площадь круга

**Пример вызова:**
area(5) - возвращает 78.53981633974483

#### perimeter(r)
Вычисляет длинну окружности.

**Параметры:** 
- r (float) - радиус круга

**Возвращает:**
- float - длинну окружности.

**Пример вызова:**
perimeter(5) - возвращает 31.41592653589793

### Модуль square.py

#### area(а)
Вычисляет площадь квадрата.

**Параметры:**
- а (float) - сторона квадрата

**Возвращает:**
- float - площадь квадрата

**Пример вызова:**
area(5.5) - возвращает 30.25

#### perimeter(r)
Вычисляет периметр квадрата.

**Параметры:** 
- a (float) - сторона квадрата

**Возвращает:**
- float - периметр квадрата.

**Пример вызова:**
perimeter(4) - возвращает 20

### Модуль triangle.py

#### area(a, h)
Вычисляет площадь треугольника.

**Параметры:**
- а(float) - сторона треугольника, h(float) - высота, опущенная на нее

**Возвращает:**
- float - площадь треугольника

**Пример вызова:**
area(5, 2) - возвращает 5.0

#### perimeter(a, b, c)
Вычисляет периметр треугольника.

**Параметры:** 
- а, b, c(float) - стороны треугольника

**Возвращает:**
- float - периметр треугольника.

**Пример вызова:**
perimetr(5, 3, 2) - возвращает 10

### Модуль square.py

#### area(а, b)
Вычисляет площадь прямоугольника.

**Параметры:**
- а (float) - сторона ширина прямоугольника, b (float) - высота прямоугольника

**Возвращает:**
- float - площадь прямоугольника

**Пример вызова:**
area(5, 3) - возвращает 15

#### area(а, b)
Вычисляет периметр прямоугольника.

**Параметры:**
- а (float) - сторона ширина квадрата, b (float) - высота прямоугольника

**Возвращает:**
- float - площадь прямоугольника

**Пример вызова:**
perimetr(4, 3) - возвращает 14

## Unit-тесты

Проект включает unit-тесты для всех модулей, написанные с использованием стандартной библиотеки unittest.

### Структура тестов
- `test_square.py` - тесты для квадрата
- `test_circle.py` - тесты для круга  
- `test_triangle.py` - тесты для треугольника
- `test_rectangle.py` - тесты для прямоугольника

### Запуск тестов

#### Запуск тестов для конкретного модуля:
```bash
python -m unittest test_square.py
python -m unittest test_circle.py
python -m unittest test_triangle.py
python -m unittest test_rectangle.py
