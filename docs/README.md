# Geometric Lib

## Общее описание решения

Эта библиотека предоставляет функции для вычисления площади и периметра различных геометрических фигур. На данный момент поддерживаются круг и квадрат.

Библиотека реализована на Python и состоит из двух модулей:
- `circle.py` - функции для работы с кругом
- `square.py` - функции для работы с квадратом

## Описание функций

### Модуль circle.py

#### area(r)
Вычисляет площадь круга.

**Параметры:**
- `r` (float) - радиус круга

**Возвращает:**
- float - площадь круга

**Пример вызова:**
area(5) - возвращает 78.53981633974483

#### perimeter(r)
Вычисляет длинну окружности.

**Параметры:** 
- `r` (float) - радиус круга

**Возвращает:**
- float - длинну окружности.

**Пример вызова:**
area(5) - возвращает 31.41592653589793

### Модуль square.py

#### area(а)
Вычисляет площадь квадрата.

**Параметры:**
- `а` (float) - сторона квадрата

**Возвращает:**
- float - площадь квадрата

**Пример вызова:**
area(5.5) - возвращает 30.25
